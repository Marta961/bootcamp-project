# Laboratorio TypeScript + React

Este directorio agrupa los módulos del bootcamp:

- `modulo-1/`: estadística en TypeScript estricto y ejecución con `tsx`.
- `modulo-2/`: modelado de dominio, unión discriminada, cliente API genérico y documentación.
- `../react/`: interfaz React + TypeScript (Vite) con tabla genérica y utilidades de fechas.

Cada subcarpeta tiene su propio `package.json` y se instala con `npm install` dentro de la carpeta correspondiente.
