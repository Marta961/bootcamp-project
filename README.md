**TaskFlow - Gestión de Tareas**

**Características:**
La interfaz de TaskFlow incluye los siguientes apartados de arriba hacia abajo:
- **Título:** TaskFlow Gestión de tareas
- **Listado de tareas:** Los usuarios podrán escribir las tareas que necesitan realizar y si están hechas
- **Estadísticas:** Se indica el número de tareas completadas y de tareas pendientes
- **Formulario:** Los usuarios podrán facilitar sugerencias y mejoras que crean convenientes

**Pruebas:** 
Se comprueva que los botones funcionan correctamente: 
- Me permite eliminar las tareas
- Generar nuevas tareas 
- Las tareas persisten después de actualizar la página web
- No me deja introducir tareas si está vacio el campo
- Se me permite encontrar tareas en el buscador
